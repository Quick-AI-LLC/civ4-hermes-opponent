# Civ4 Player Skill — Reference Notes

## Sources

### Reddit: Civ4 2023 Mini-Guide (r/CivIV)
URL: https://www.reddit.com/r/CivIV/comments/10j2bbr/civ4_2023_miniguide_for_new_and_returning_players/

Key mechanics extracted for the skill's system prompt:

- **Big Fat Cross (BFC)**: Cities work a 5x5 grid (2 cardinal, 1 diagonal). Worker improvement priority inside BFC: food resources -> production mines -> gold cottages.
- **Whipping (Slavery)**: 1 population = 30 hammers at normal speed. Sacrifice food-surplus citizens to rush-build settlers, workers, key military units. Accept pop loss and temporary unhappiness.
- **Cottage Economy**: Cottages grow in value the longer they're worked uninterrupted. Place on grassland/plains, grow through specialist turns, upgrade to villages/towns for massive late-game gold.
- **Expansion Timing**: Get 3 cities fast, expand to ~6. First half-dozen box out rivals for resources and land. Each new city adds upkeep pressure.
- **National Wonders**: 2 per city, 14 empire-wide. Plan city specialization chains for efficient wonder pairing.
- **War Mechanics**: Need any 2 of 3 to win - more units, better tech, or patience. Defensive bonuses are very powerful. Stack siege + collateral damage units together.
- **Spy Usage**: Sabotage enemy production before offensives, steal tech from leaders.
- **Difficulty Scaling**: Lower difficulties allow faster expansion and AI exploitation; higher difficulties demand efficiency and defensive play.
